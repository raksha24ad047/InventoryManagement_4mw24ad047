# Inventory Management System

## Project Overview

This project is a full-stack Inventory Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

The system helps manage products, suppliers, warehouses, stock, purchases, and sales through a web-based interface. It also demonstrates the use of SQL queries, stored procedures, and triggers.

---
